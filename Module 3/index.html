HTML & CSS Basic Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content, struckture and meaning. you don't used it for styling purposes. Use css fot that insted!

  • HTML uses "elements" to describe (annotate) content

    • HTML elements typically have an opening tag and a clodsing tag

  • You can also have void (empty) elements like images
  • You can also configure elements with attributes

    • There's a long list available elements but you'll gain experience over time, no worries.

Learn more about all available HTML elements on the MDN HTML element reference.
